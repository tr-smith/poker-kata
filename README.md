# poker-kata

## Background
This kata simulates a hand dealt in five card poker.

The user is dealt 5 cards from a shuffled deck.

The Possible Values are:

- Royal Flush
- Straight Flush
- Four of a kind
- Full House
- Flush
- Straight
- Three of a kind
- Two pair
- One pair
- High Card

## Feature
*As a user<br>
I want to see what poker value my hand is worth as dealt.
